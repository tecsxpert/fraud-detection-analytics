# Presentation: Enterprise Analytics Pro

## Slide 1: Problem Statement
- **Fragmented Data**: Organizations struggle with silos across Software, Hardware, and Services.
- **Delayed Insights**: Manual reporting takes days; business opportunities are missed.
- **Complexity**: Monitoring real-time system health requires specialized tools that don't talk to each other.

---

## Slide 2: System Architecture
- **Vite & React 19**: Ultra-fast, type-safe frontend with modern hooks and transitions.
- **Spring Boot (Simulated)**: RESTful API handling pagination, sorting, and stateful management.
- **Gemini AI Integration**: Context-aware project analysis and intelligent assistant.
- **SSE Streaming**: Real-time event monitoring via Server-Sent Events pattern.

---

## Slide 3: Demo Flow
1. **Secure Access**: JWT-based login entry.
2. **Bird's Eye View**: Executive dashboard with Recharts KPIs.
3. **Operational Deep-Dive**: Project management list with AI-powered health analysis.
4. **Live Pulse**: Real-time streaming monitor for system integrity.

---

# Demo Scenarios

### Scenario 1: Executive Overview
- **Step**: Login and view the Dashboard.
- **Expectation**: Responsive charts show revenue distribution and growth targets at a glance.

### Scenario 2: Project Life-cycle Management
- **Step**: Create a new project, edit its status to 'Active'.
- **Expectation**: Form validation prevents invalid entries; list view updates instantly.

### Scenario 3: AI-Powered Intelligence
- **Step**: Open Project Detail and click "Analyze Now".
- **Expectation**: Gemini AI provides specific business risks and recommendations based on the project score and value.

### Scenario 4: Strategic Analytics
- **Step**: Navigate to Analytics and filter by '1Y'.
- **Expectation**: Regression lines and status distribution adjust to reflect long-term trends.

### Scenario 5: Real-time System Audit
- **Step**: Start the Stream feed in the Streaming page.
- **Expectation**: A live terminal-style log shows system events as they happen, simulating backend sync.
